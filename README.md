# 📖 About

**ServiceSphere** is a full-stack web application designed to bridge the gap between customers and trusted local service providers.

Whether someone needs an electrician, plumber, mechanic, cleaner, tutor, carpenter, or any other local professional, ServiceSphere offers a seamless platform to discover, compare, book, and review services.

The platform provides secure authentication, intuitive dashboards, booking management, reviews, and a responsive user interface to ensure a smooth experience across all devices.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Nagacharan/ServiceSphere.git
```

```bash
cd ServiceSphere
```

---

## Install Dependencies

```bash
npm install
```

or

```bash
npm run install-all
```

---

## Run the Project

```bash
npm run dev
```

---

Frontend

```
http://localhost:5173
```

Backend

```
http://localhost:5000
```
